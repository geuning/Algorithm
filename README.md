# Algorithm
알고리즘 문제 풀이 기록

## 커밋 컨벤션
### Format

#### {Tag}: {#Issue} {출처} {난이도} {제목(번호)}
- Issues탭에 문제와 관련된 유형 추가
- 문제 Commit 시 관련 이슈 넘버 태그 
- Ex) solve: #7 백준 S1 하노이 탑 이동 순서(11729)

### Tag
- `solve`: 풀이에 성공한 새 문제 추가
- `fail`: 풀이에 실패한 새 문제 추가
- `improve`: 새로운 성공 풀이를 추가한 문제
- `docs`: readme 등 문서 수정
